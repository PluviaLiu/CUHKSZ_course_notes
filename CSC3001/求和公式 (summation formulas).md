关于换元Substitution：

#### 换元的逻辑链条

你现在手上有的是：

$$\sum_{k=2}^{n} \binom{n-2}{k-2}$$

**目标**：把这个"看起来别扭的 $k-2$"换成一个"从0开始的新变量"，方便套二项式定理。

---

- ● **定义新变量**：令 $j = k - 2$。
    - 这个代换的意思是：我不再用 $k$ 来数，我改用 $j = k-2$ 来数。

- ● **改上下限**：
    - 原来 $k$ 的下限是 $k=2$ → 代入得 $j = 2-2 = 0$，所以新下限是 $j=0$。
    - 原来 $k$ 的上限是 $k=n$ → 代入得 $j = n-2$，所以新上限是 $j=n-2$。

- ● **改组合数里的下标**：
    - 原来的下标是 $k-2$。
    - 因为 $j = k-2$，所以 $k-2$ 直接换成 $j$。
    - $\binom{n-2}{k-2}$ 变成 $\binom{n-2}{j}$。

- ● **得到新的求和式**：

$$\sum_{k=2}^{n} \binom{n-2}{k-2} = \sum_{j=0}^{n-2} \binom{n-2}{j}$$

---

#### 工科直觉：为什么这两个求和完全一样？

你可以把它理解为**重新贴标签**。

原来的求和列出来是：

| $k$ | 组合数 |
| --- | --- |
| $k=2$ | $\binom{n-2}{0}$ |
| $k=3$ | $\binom{n-2}{1}$ |
| $k=4$ | $\binom{n-2}{2}$ |
| ... | ... |
| $k=n$ | $\binom{n-2}{n-2}$ |

换元后列出来是：

| $j$ | 组合数 |
| --- | --- |
| $j=0$ | $\binom{n-2}{0}$ |
| $j=1$ | $\binom{n-2}{1}$ |
| $j=2$ | $\binom{n-2}{2}$ |
| ... | ... |
| $j=n-2$ | $\binom{n-2}{n-2}$ |

**两张表的右边完全一样**，只是左边的标签从 $k$ 换成了 $j$。所以两个求和式的值当然相等。

---

#### 最后一步：为什么等于 $2^{n-2}$？

换元之后得到：

$$\sum_{j=0}^{n-2} \binom{n-2}{j}$$

这正好是把 $\binom{n-2}{0} + \binom{n-2}{1} + \dots + \binom{n-2}{n-2}$ **全部加起来**，即对 $n-2$ 阶二项式系数求完整的和。

由二项式定理 (Binomial Theorem)，令 $x=1$：

$$(1+1)^{n-2} = \sum_{j=0}^{n-2}\binom{n-2}{j} = 2^{n-2}$$

所以直接得到 $2^{n-2}$。
