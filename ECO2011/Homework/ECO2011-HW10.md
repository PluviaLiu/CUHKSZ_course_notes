# Q1
这道微观经济学题目主要考察垄断者的利润最大化决策、社会福利分析以及政府价格管制（价格上限）的影响。以下是详细的解答过程：

### (a) 曲线计算与绘图

首先，根据给出的成本函数和需求函数进行推导：
[[微观经济学连招计算公式]]
- **总成本 (TC):** $TC = \frac{1}{2}Q^2$（由总固定成本TFC和总可变成本TVC组成，其中固定成本是常数项，没有常数项就是没有固定成本）
- **边际成本 (MC):** $MC = \frac{dTC}{dQ} = Q$
- **平均总成本 (ATC):** $ATC = \frac{TC}{Q} = \frac{1}{2}Q$
- **平均可变成本 (AVC):** 由于没有固定成本，$AVC = ATC + AFC（平均固定成本） = \frac{1}{2}Q$
- **需求曲线 (Demand):** $Q = 120 - P \implies P = 120 - Q$
- **总收益 (TR):** $TR = P \cdot Q = (120 - Q)Q = 120Q - Q^2$
- **边际收益 (MR):** $MR = \frac{dTR}{dQ} = 120 - 2Q$

**绘图说明：**
- **Demand:** 纵轴截距 120，横轴截距 120，斜率为 -1。
- **MR:** 纵轴截距 120，横轴截距 60，斜率为 -2（比需求曲线更陡）。
- **MC:** 从原点出发，斜率为 1 的射线。
- **ATC/AVC:** 从原点出发，斜率为 0.5 的射线（位于 MC 下方）。

---

### (b) 利润最大化决策与福利分析
[[利润最大化的条件]]

**1. 利润最大化条件：$MR = MC$**
$120 - 2Q = Q \implies 3Q = 120 \implies Q^* = 40$
将 $Q^*$ 代入需求方程：$P^* = 120 - 40 = 80$

**2. 福利计算：**
- **消费者剩余 (CS):** 需求曲线下方、价格线 $P=80$ 上方的三角形面积。
  $CS = \frac{1}{2} \times (120 - 80) \times 40 = 800$
- **生产者剩余 (PS):** 价格线 $P=80$ 下方、MC 曲线（即供给曲线）上方的面积。
  $PS = \int_0^{40} (80 - Q) dQ = [80Q - \frac{1}{2}Q^2]_0^{40} = 3200 - 800 = 2400$
- **总剩余 (TS):** $TS = CS + PS = 800 + 2400 = 3200$
- **无谓损失 (DWL):**
  社会最优产量发生在 $P = MC$ 时：$120 - Q = Q \implies Q_{eff} = 60$。
  $DWL = \frac{1}{2} \times (P^* - MC|_{Q=40}) \times (Q_{eff} - Q^*) = \frac{1}{2} \times (80 - 40) \times (60 - 40) = 400$

### (c) 价格上限 $P = 70$ 的影响

**1. 垄断者的决策：**
当政府设定 $P = 70$ 的价格上限时，垄断者在 $Q \le 50$（因为 $120-50=70$）时的边际收益变为水平线 $MR = 70$。
由于在 $Q=50$ 时，$MC = 50 < MR = 70$，垄断者会尽可能多地生产以满足该价格下的需求。
- **产量:** $Q = 50$
- **价格:** $P = 70$

**2. 利润与福利计算：**
- **利润 ($\pi$):** $TR - TC = (70 \times 50) - \frac{1}{2}(50)^2 = 3500 - 1250 = 2250$
- **CS:** $\frac{1}{2} \times (120 - 70) \times 50 = 1250$
- **PS:** $\int_0^{50} (70 - Q) dQ = [70Q - \frac{1}{2}Q^2]_0^{50} = 3500 - 1250 = 2250$
- **DWL:** 此时产量更接近社会最优量 (60)。
  $DWL = \frac{1}{2} \times (P|_{Q=50} - MC|_{Q=50}) \times (60 - 50) = \frac{1}{2} \times (70 - 50) \times 10 = 100$

**为什么 DWL 不同？**
价格上限削弱了垄断者的市场力量，迫使其面临一条水平的边际收益曲线（在上限范围内）。这降低了垄断者通过限制产量来抬高价格的动机，使产量增加，从而减少了由于产量不足导致的社会福利损失。

---

### (d) 最大化总剩余的价格上限

要最大化总剩余，政府应将价格上限设定在**社会最优水平**，即需求曲线与边际成本曲线的交点。

**1. 设定价格上限：**
$P = MC \implies 120 - Q = Q \implies Q = 60$
代入需求方程：$P = 120 - 60 = 60$
- **价格上限:** $P_c = 60$

**2. 结果分析：**
- **产量:** $Q = 60$
- **利润:** $\pi = TR - TC = (60 \times 60) - \frac{1}{2}(60)^2 = 3600 - 1800 = 1800$
- **无谓损失 (DWL):** $0$（因为此时达到了社会最优产量，资源配置完全有效）。
