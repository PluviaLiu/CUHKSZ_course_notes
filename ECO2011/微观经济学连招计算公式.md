
# 📊 微观经济学核心计算连招表

## 第一阶段：成本拆解 (Cost Analysis)
拿到 $TC$（总成本）函数后的第一步：

1.  **总固定成本 (TFC):** $TC$ 中不含 $Q$ 的常数项。
2.  **总可变成本 (TVC):** $TC$ 中含有 $Q$ 的所有项 ($TVC = TC - TFC$)。
3.  **边际成本 (MC):** $MC = \frac{dTC}{dQ}$ （对 $TC$ 求导）。
4.  **平均总成本 (ATC):** $ATC = \frac{TC}{Q}$。
5.  **平均可变成本 (AVC):** $AVC = \frac{TVC}{Q}$。
6.  **平均固定成本 (AFC):** $AFC = \frac{TFC}{Q}$。
    *   **💡 连招技巧：** 若 $TC$ 没有常数项，则 $TFC=0$，此时 $ATC = AVC$。

---

## 第二阶段：收益与需求 (Revenue & Demand)
拿到 $P$（需求曲线）或 $Q$（需求量）后的处理：

1.  **反需求函数 (Inverse Demand):** 确保方程写成 $P = f(Q)$ 的形式（例如 $P = a - bQ$）。
2.  **总收益 (TR):** $TR = P \times Q = (a - bQ)Q = aQ - bQ^2$。
3.  **边际收益 (MR):** $MR = \frac{dTR}{dQ}$。
    *   **💡 连招技巧（线性需求倍速法）：** 若 $P = a - bQ$，则 $MR = a - 2bQ$（截距不变，斜率翻倍）。
4.  **平均收益 (AR):** $AR = \frac{TR}{Q} = P$（平均收益永远等于价格）。

---

## 第三阶段：利润最大化 (Profit Maximization)
所有企业的终极目标：

1.  **通用条件：** $MR = MC$。
2.  **计算步骤：**
    *   第一步：解出 $MR = MC$ 得到最优产量 $Q^*$。
    *   第二步：将 $Q^*$ 代入**需求曲线** $P = f(Q)$ 得到最优价格 $P^*$。
    *   第三步：计算利润 $\pi = TR - TC$ 或 $\pi = (P - ATC) \times Q$。

---

## 第四阶段：社会福利与效率 (Welfare & Efficiency)
评估市场好坏的标准：

1.  **消费者剩余 (CS):** 需求曲线下方，价格线上方的面积。
    *   $CS = \int_0^{Q^*} (P_{demand} - P^*) dQ$ （三角形面积：$\frac{1}{2} \times Q^* \times (P_{max} - P^*)$）。
2.  **生产者剩余 (PS):** 价格线下方，MC 曲线（供给曲线）上方的面积。
    *   $PS = \int_0^{Q^*} (P^* - MC) dQ$。
    *   **💡 关系式：** $PS = 利润 + TFC$。
3.  **总剩余 (TS):** $TS = CS + PS$。
4.  **社会最优产量 ($Q_{eff}$):** 发生在 $P = MC$ 时（即完全竞争均衡点）。
5.  **无谓损失 (DWL):** 社会最优总剩余与当前实际总剩余的差额。
    *   图形上通常是 $Q^*$ 到 $Q_{eff}$ 之间，夹在 $D$ 和 $MC$ 之间的那个三角形。

---

## 第五阶段：政府干预 (Government Intervention)

1.  **价格上限 (Price Ceiling):** 
    *   若 $P_{cap} < P^*$, 垄断者被迫按 $P_{cap}$ 定价。
    *   新的 $MR$ 在达到需求曲线前是一条水平线 $MR = P_{cap}$。
2.  **税收 (Tax):**
    *   **从量税 (t):** 新的 $MC' = MC + t$。
    *   **利润税 (Lump-sum tax):** 只影响利润 $\pi$，不影响 $MC$ 和 $MR$，因此不影响产量 $Q$ 和价格 $P$。
