采用**“四步结构法”**（Definition - Elaboration - Example - Synthesis），帮我梳理这个lec中涉及的定义。请务必全面。

#### 1. Definition（精准定义）：直击核心
- **逻辑：** 用一句话给出该术语的学术定义。
- **要点：** 必须包含**关键词（Keywords）**。如果是翻译过来的术语，最好附带英文原词。
- **避坑：** 不要用“我觉得……”或者太口语化的表达。

#### 2. Elaboration（原理解析/补充说明）：展示深度
- **逻辑：** 解释这个概念为什么存在，或者它的运作机制是什么。
- **要点：** 谈谈它的成因、分类或对立面。这能体现你对知识点的掌握是立体的。

#### 3. AI-related Example（AI相关案例）：具体化
- **逻辑：** 引入一个具体的、真实的AI案例。
- **要点：** 案例要**具体**（具体的模型、具体的公司、具体的事件），不要泛泛而谈“现在的机器人”。
- **推荐：** 引用我们之前讨论过的经典案例（如 GPT-4, COMPAS系统, 或者 Uber 的算法管理）。

#### 4. Synthesis/Analysis（逻辑闭环/分析）：防止丢分的关键
- **逻辑：** 强力连接——说明这个案例**如何**体现了上述定义。
- **要点：** 很多学生只写了定义和例子，但没有解释两者之间的关系。这一步是告诉老师：“你看，这个例子里的 A 现象正好对应了定义里的 B 属性。”
