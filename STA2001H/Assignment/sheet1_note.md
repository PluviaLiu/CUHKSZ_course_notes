# Q1

#### 问题翻译
掷两个六面骰子，求两个骰子点数之和等于7的概率。

#### 背景条件解释
**基本概念**：
- **样本空间 Sample Space** 所有可能结果的集合，用 $\Omega$ 表示
- **事件 Event** $\Omega$ 的子集，表示某个可能发生的结果集合
- **概率 Probability** 事件发生的可能性大小，用 $P$ 表示

**具体条件**：
- 每个骰子有6个面，点数为1,2,3,4,5,6
- 两个骰子独立（independent）掷出
- 我们关心的事件是"两个骰子点数之和等于7"

#### 解答思路

**步骤1：确定样本空间**
- 因为第一个骰子有6种可能，第二个骰子有6种可能
- 总共有 $6 \times 6 = 36$ 种等可能的结果
- 样本空间 $\Omega$ 有36个元素

**步骤2：找出有利事件**
- 我们需要找出所有 $(x,y)$ 使得 $x + y = 7$
- 可能的组合有：
  - $(1,6)$：第一个骰子1，第二个骰子6
  - $(2,5)$：第一个骰子2，第二个骰子5  
  - $(3,4)$：第一个骰子3，第二个骰子4
  - $(4,3)$：第一个骰子4，第二个骰子3
  - $(5,2)$：第一个骰子5，第二个骰子2
  - $(6,1)$：第一个骰子6，第二个骰子1
- 总共有6种有利结果

**步骤3：计算概率**
- 基于古典概型（Classical Probability）
- $P(\text{事件}) = \frac{\text{有利结果数}}{\text{总结果数}}$
- 所以 $P(x + y = 7) = \frac{6}{36} = \frac{1}{6}$

#### 关键概念总结
- **样本空间 Sample Space** 所有可能结果的集合
- **事件 Event** 我们关心的某个结果集合
- **概率 Probability** 事件发生的可能性度量
- **独立 Independent** 一个事件的发生不影响另一个事件

# Q2

#### 问题翻译
已知一个家庭有两个孩子，其中至少有一个是女孩。求两个孩子都是女孩的概率。

#### 背景条件解释
**基本概念**：
- **条件概率 Conditional Probability** 在已知某个事件发生的条件下，另一个事件发生的概率
- **独立 Independent** 两个事件的发生互不影响
- **互斥 Mutually Exclusive** 两个事件不能同时发生

**具体条件**：
- 一个家庭有两个孩子
- 每个孩子可能是男孩（Boy）或女孩（Girl）
- 已知条件：至少有一个女孩
- 求解条件：两个孩子都是女孩的概率

#### 解答思路

**步骤1：确定样本空间**
- 每个孩子有两种可能：B（男孩）或G（女孩）
- 两个孩子的所有可能结果：
  - $(B,B)$：两个都是男孩
  - $(B,G)$：第一个男孩，第二个女孩
  - $(G,B)$：第一个女孩，第二个男孩  
  - $(G,G)$：两个都是女孩
- 样本空间 $\Omega = \{(B,B), (B,G), (G,B), (G,G)\}$

**步骤2：应用已知条件**
- 已知条件：至少有一个女孩
- 这意味着 $(B,B)$ 不可能发生
- 剩下的可能结果：$(B,G), (G,B), (G,G)$
- 条件样本空间有3个元素

**步骤3：计算条件概率**
- 条件概率公式：$P(A|B) = \frac{P(A \cap B)}{P(B)}$
- 这里事件 $A$ = "两个孩子都是女孩"
- 事件 $B$ = "至少有一个女孩"
- $P(A \cap B) = P(\text{两个女孩}) = \frac{1}{4}$
- $P(B) = P(\text{至少一个女孩}) = \frac{3}{4}$
- 所以 $P(A|B) = \frac{\frac{1}{4}}{\frac{3}{4}} = \frac{1}{3}$

#### 关键概念总结
- **条件概率 Conditional Probability** 在给定条件下的概率
- **样本空间 Sample Space** 所有可能结果的集合
- **古典概型 Classical Probability** 等可能结果的概率计算
