# Q1

#### 问题翻译
某种疾病在人群中的发病率为0.1%。检测这种疾病的测试有如下特性：
- 如果患者有病，测试结果为阳性的概率是99%
- 如果患者没病，测试结果为阳性的概率是5%
现在某人测试结果为阳性，求他确实患有这种疾病的概率。

#### 背景条件解释
**基本概念**：
- **先验概率 Prior Probability** 在获得新信息之前的概率
- **似然概率 Likelihood** 在给定条件下观察到的概率
- **后验概率 Posterior Probability** 在获得新信息后的概率
- **贝叶斯定理 Bayes' Theorem** 更新概率的公式

**具体条件**：
- 疾病发病率 $P(D) = 0.001$ （先验概率）
- 测试准确性：
  - 真阳性率 $P(T^+|D) = 0.99$ （灵敏度 Sensitivity）
  - 假阳性率 $P(T^+|\neg D) = 0.05$ （1 - 特异性 Specificity）
- 测试结果：阳性 $T^+$
- 求解：后验概率 $P(D|T^+)$

#### 解答思路

**步骤1：明确已知和求解**
- 已知：
  - $P(D) = 0.001$ （疾病先验概率）
  - $P(T^+|D) = 0.99$ （患者测试阳性概率）
  - $P(T^+|\neg D) = 0.05$ （健康人测试阳性概率）
- 求解：$P(D|T^+)$ （测试阳性时真患病概率）

**步骤2：应用贝叶斯定理**
- 贝叶斯定理公式：
  $$P(D|T^+) = \frac{P(T^+|D) \cdot P(D)}{P(T^+)}$$
- 分子：$P(T^+|D) \cdot P(D) = 0.99 \times 0.001 = 0.00099$

**步骤3：计算分母（全概率）**
- 使用全概率公式：
  $$P(T^+) = P(T^+|D) \cdot P(D) + P(T^+|\neg D) \cdot P(\neg D)$$
- 计算：
  - $P(\neg D) = 1 - P(D) = 1 - 0.001 = 0.999$
  - $P(T^+) = (0.99 \times 0.001) + (0.05 \times 0.999) = 0.00099 + 0.04995 = 0.05094$

**步骤4：计算后验概率**
- $P(D|T^+) = \frac{0.00099}{0.05094} \approx 0.0194$
- 约1.94%

#### 关键概念总结
- **贝叶斯定理 Bayes' Theorem** $P(A|B) = \frac{P(B|A)P(A)}{P(B)}$
- **先验概率 Prior Probability** 事件原本的概率
- **后验概率 Posterior Probability** 获得新信息后的概率
- **全概率公式 Total Probability** $P(B) = \sum P(B|A_i)P(A_i)$

# Q2

#### 问题翻译
一个袋子中有5个红球和3个蓝球。随机抽取两个球，求：
1. 两个都是红球的概率
2. 一个红球一个蓝球的概率
3. 两个都是蓝球的概率

#### 背景条件解释
**基本概念**：
- **组合 Combination** 从一组物品中选取一定数量的方法数
- **排列 Permutation** 考虑顺序的选取方法数
- **超几何分布 Hypergeometric Distribution** 无放回抽样的分布
- **概率 Probability** 事件发生的可能性大小

**具体条件**：
- 袋子中有5个红球（R）和3个蓝球（B），总共8个球
- 随机抽取两个球，无放回（without replacement）
- 求解不同组合的概率

#### 解答思路

**步骤1：确定总的抽取方式**
- 从8个球中抽取2个的组合数：
  $$\binom{8}{2} = \frac{8!}{2!6!} = \frac{8 \times 7}{2 \times 1} = 28$$
- 总共有28种等可能的抽取结果

**步骤2：计算两个都是红球的概率**
- 从5个红球中抽取2个的组合数：
  $$\binom{5}{2} = \frac{5!}{2!3!} = \frac{5 \times 4}{2 \times 1} = 10$$
- 概率：$P(\text{两个红球}) = \frac{10}{28} = \frac{5}{14}$

**步骤3：计算一个红球一个蓝球的概率**
- 从5个红球中取1个，从3个蓝球中取1个：
  $$\binom{5}{1} \times \binom{3}{1} = 5 \times 3 = 15$$
- 概率：$P(\text{一红一蓝}) = \frac{15}{28}$

**步骤4：计算两个都是蓝球的概率**
- 从3个蓝球中抽取2个的组合数：
  $$\binom{3}{2} = \frac{3!}{2!1!} = 3$$
- 概率：$P(\text{两个蓝球}) = \frac{3}{28}$

**步骤5：验证概率总和**
- $\frac{5}{14} + \frac{15}{28} + \frac{3}{28} = \frac{10}{28} + \frac{15}{28} + \frac{3}{28} = \frac{28}{28} = 1$
- 验证正确

#### 关键概念总结
- **组合 Combination** $\binom{n}{k} = \frac{n!}{k!(n-k)!}$
- **超几何分布 Hypergeometric Distribution** 无放回抽样的概率分布
- **样本空间 Sample Space** 所有可能结果的集合
- **概率 Probability** 事件发生的可能性度量
