# Q1

#### 问题翻译
一个篮球运动员每次投篮的命中率为0.7，投篮10次，求恰好命中8次的概率。

#### 背景条件解释
**基本概念**：
- **二项分布 Binomial Distribution** 进行n次独立伯努利试验的成功次数分布
- **伯努利试验 Bernoulli Trial** 只有两种结果的独立重复试验
- **参数 Parameters** n=试验次数，p=每次成功概率

**具体条件**：
- 试验次数 $n = 10$ （投篮10次）
- 成功概率 $p = 0.7$ （每次投篮命中率）
- 失败概率 $q = 1 - p = 0.3$ （每次投篮不中概率）
- 独立性：每次投篮结果相互独立（independent）
- 求解：恰好命中8次的概率

#### 解答思路

**步骤1：识别分布类型**
- 这是典型的二项分布问题
- 满足二项分布的条件：
  - 固定次数试验：n=10
  - 每次试验只有两种结果：命中/不中
  - 每次试验成功概率相同：p=0.7
  - 试验相互独立：投篮结果互不影响

**步骤2：应用二项分布公式**
- 二项分布概率质量函数（PMF）：
  $$P(X = k) = \binom{n}{k} p^k q^{n-k}$$
- 其中：
  - $\binom{n}{k}$ 是组合数，表示从n次中选择k次成功的方式数
  - $\binom{n}{k} = \frac{n!}{k!(n-k)!}$

**步骤3：计算具体数值**
- 代入数值：$n=10$, $k=8$, $p=0.7$, $q=0.3$
- 计算组合数：
  $$\binom{10}{8} = \frac{10!}{8!2!} = \frac{10 \times 9}{2 \times 1} = 45$$
- 计算概率：
  $$P(X = 8) = 45 \times (0.7)^8 \times (0.3)^2$$

**步骤4：最终计算**
- $(0.7)^8 = 0.05764801$
- $(0.3)^2 = 0.09$
- $P(X = 8) = 45 \times 0.05764801 \times 0.09 = 0.233474$

#### 关键概念总结
- **二项分布 Binomial Distribution** $P(X=k) = \binom{n}{k} p^k q^{n-k}$
- **组合数 Combination** $\binom{n}{k} = \frac{n!}{k!(n-k)!}$
- **独立 Independent** 试验之间互不影响
- **伯努利试验 Bernoulli Trial** 只有两种结果的试验

# Q2

#### 问题翻译
某工厂生产的产品次品率为2%。随机抽取20个产品，求：
1. 没有次品的概率
2. 恰好有2个次品的概率
3. 至少有1个次品的概率

#### 背景条件解释
**基本概念**：
- **二项分布 Binomial Distribution** n次独立试验的成功次数分布
- **次品率 Defect Rate** 产品不合格的概率
- **样本 Sample** 从总体中抽取的部分个体
- **概率分布 Probability Distribution** 随机变量的概率规律

**具体条件**：
- 次品率 $p = 0.02$ （每个产品是次品的概率）
- 正品率 $q = 1 - p = 0.98$ （每个产品是正品的概率）
- 样本量 $n = 20$ （抽取20个产品）
- 独立性：产品生产相互独立
- 求解：不同情况的概率

#### 解答思路

**步骤1：确定分布**
- 这是一个二项分布问题：$X \sim B(n=20, p=0.02)$
- 概率质量函数：$P(X = k) = \binom{20}{k} (0.02)^k (0.98)^{20-k}$

**步骤2：计算没有次品的概率**
- $P(X = 0) = \binom{20}{0} (0.02)^0 (0.98)^{20}$
- $\binom{20}{0} = 1$
- $(0.02)^0 = 1$
- $(0.98)^{20} \approx 0.6676$
- $P(X = 0) = 1 \times 1 \times 0.6676 = 0.6676$

**步骤3：计算恰好有2个次品的概率**
- $P(X = 2) = \binom{20}{2} (0.02)^2 (0.98)^{18}$
- $\binom{20}{2} = \frac{20 \times 19}{2 \times 1} = 190$
- $(0.02)^2 = 0.0004$
- $(0.98)^{18} \approx 0.6951$
- $P(X = 2) = 190 \times 0.0004 \times 0.6951 \approx 0.0528$

**步骤4：计算至少有1个次品的概率**
- 使用互补事件：
  $P(X \ge 1) = 1 - P(X = 0)$
- $P(X \ge 1) = 1 - 0.6676 = 0.3324$

#### 关键概念总结
- **二项分布 Binomial Distribution** $P(X=k) = \binom{n}{k} p^k q^{n-k}$
- **互补事件 Complementary Event** $P(A) = 1 - P(A^c)$
- **独立性 Independent** 事件之间互不影响
- **样本量 Sample Size** $n$ 试验或观察的次数
