# Q1

#### 问题翻译
某医院平均每小时有5个急诊病人到达，求在一小时内恰好有3个病人到达的概率。

#### 背景条件解释
**基本概念**：
- **泊松分布 Poisson Distribution** 单位时间内稀有事件发生次数的分布
- **参数λ Lambda** 单位时间内事件发生的平均次数
- **平稳性 Stationary** 事件发生概率在时间上保持恒定
- **独立性 Independent** 事件发生相互独立

**具体条件**：
- 平均到达率 $\lambda = 5$ 个/小时
- 时间区间：1小时
- 事件满足泊松过程条件：
  - 平稳性：每小时平均5个病人
  - 独立性：病人到达相互独立
  - 稀有性：在很小时间区间内最多一个病人到达
- 求解：恰好3个病人到达的概率

#### 解答思路

**步骤1：确认分布适用性**
- 检查是否符合泊松分布的条件：
  - 事件是离散的（病人个数）
  - 事件发生的平均次数已知（λ=5）
  - 事件之间相互独立
  - 在小时间区间内最多发生一次事件
- 符合泊松分布的特征

**步骤2：应用泊松分布公式**
- 泊松分布概率质量函数：
  $$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}$$
- 其中：
  - $\lambda$ 是单位时间内的平均发生次数
  - $k$ 是实际发生次数
  - $e$ 是自然常数（约2.71828）
  - $k!$ 是k的阶乘

**步骤3：代入数值计算**
- 参数：$\lambda = 5$, $k = 3$
- 计算各项：
  - $\lambda^k = 5^3 = 125$
  - $e^{-\lambda} = e^{-5} \approx 0.006737947$
  - $k! = 3! = 6$

**步骤4：计算最终概率**
- $P(X = 3) = \frac{125 \times 0.006737947}{6}$
- 分子：$125 \times 0.006737947 = 0.842243375$
- 分母：$6$
- $P(X = 3) = \frac{0.842243375}{6} \approx 0.140374$

#### 关键概念总结
- **泊松分布 Poisson Distribution** $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
- **参数λ Lambda** 单位时间内的平均发生次数
- **平稳性 Stationary** 概率在时间上保持不变
- **独立性 Independent** 事件之间互不影响
- **阶乘 Factorial** $k! = k \times (k-1) \times \cdots \times 1$

# Q2

#### 问题翻译
某书店平均每天有10个顾客购买某本书。求：
1. 某天恰好有8个顾客购买的概率
2. 某天超过15个顾客购买的概率
3. 某天少于3个顾客购买的概率

#### 背景条件解释
**基本概念**：
- **泊松分布 Poisson Distribution** 单位时间内事件发生次数的分布
- **平均发生率 Rate** 单位时间内事件的平均发生次数
- **累积概率 Cumulative Probability** 多个事件的联合概率
- **互补事件 Complementary Event** 事件的反面概率

**具体条件**：
- 平均购买率 $\lambda = 10$ 个/天
- 时间区间：1天
- 事件满足泊松分布条件：
  - 平稳性：每天平均10个购买
  - 独立性：顾客购买行为相互独立
  - 稀有性：短时间内最多一个顾客购买
- 求解：不同情况的概率

#### 解答思路

**步骤1：确定分布参数**
- 使用泊松分布：$X \sim \text{Poisson}(\lambda = 10)$
- 概率质量函数：$P(X = k) = \frac{10^k e^{-10}}{k!}$

**步骤2：计算恰好8个顾客购买的概率**
- $P(X = 8) = \frac{10^8 e^{-10}}{8!}$
- $10^8 = 100000000$
- $e^{-10} \approx 4.539992976 \times 10^{-5}$
- $8! = 40320$
- $P(X = 8) = \frac{100000000 \times 4.539992976 \times 10^{-5}}{40320} \approx \frac{4539.992976}{40320} \approx 0.1126$

**步骤3：计算超过15个顾客购买的概率**
- $P(X > 15) = 1 - P(X \le 15)$
- 需要计算累积概率：
  $P(X \le 15) = \sum_{k=0}^{15} \frac{10^k e^{-10}}{k!}$
- 使用泊松分布表或计算器：
  $P(X \le 15) \approx 0.9513$
- 所以 $P(X > 15) = 1 - 0.9513 = 0.0487$

**步骤4：计算少于3个顾客购买的概率**
- $P(X < 3) = P(X \le 2) = P(X = 0) + P(X = 1) + P(X = 2)$
- 计算各项：
  - $P(X = 0) = \frac{10^0 e^{-10}}{0!} = e^{-10} \approx 4.54 \times 10^{-5}$
  - $P(X = 1) = \frac{10^1 e^{-10}}{1!} = 10e^{-10} \approx 4.54 \times 10^{-4}$
  - $P(X = 2) = \frac{10^2 e^{-10}}{2!} = \frac{100e^{-10}}{2} \approx 2.27 \times 10^{-3}$
- $P(X < 3) \approx 4.54 \times 10^{-5} + 4.54 \times 10^{-4} + 2.27 \times 10^{-3} \approx 0.00277$

#### 关键概念总结
- **泊松分布 Poisson Distribution** $P(X=k) = \frac{\lambda^k e^{-\lambda}}{k!}$
- **累积概率 Cumulative Probability** $P(X \le k) = \sum_{i=0}^{k} P(X = i)$
- **互补事件 Complementary Event** $P(X > k) = 1 - P(X \le k)$
- **参数λ Lambda** 单位时间内的平均发生次数
