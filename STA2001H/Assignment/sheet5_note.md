# Q1

#### 问题翻译
某次考试成绩服从正态分布 $N(75, 10^2)$，求成绩大于85分的概率。

#### 背景条件解释
**基本概念**：
- **正态分布 Normal Distribution** 对称的连续概率分布
- **参数 Parameters** $\mu$（均值 mean），$\sigma^2$（方差 variance）
- **标准化 Standardization** 转换为标准正态分布的过程
- **标准正态分布 Standard Normal Distribution** $N(0,1)$

**具体条件**：
- 成绩分布：$X \sim N(\mu, \sigma^2)$
- 均值：$\mu = 75$ 分
- 方差：$\sigma^2 = 10^2$，所以标准差 $\sigma = 10$ 分
- 求解：$P(X > 85)$

#### 解答思路

**步骤1：理解正态分布性质**
- 正态分布是连续型分布，概率密度函数呈钟形曲线
- 对称性：关于均值 $\mu = 75$ 对称
- 标准化：任何正态分布都可以转换为标准正态分布
- 标准化公式：$Z = \frac{X - \mu}{\sigma}$

**步骤2：进行标准化**
- 计算85分对应的Z值：
  $$Z = \frac{85 - 75}{10} = \frac{10}{10} = 1$$
- 所以 $P(X > 85) = P(Z > 1)$
- 其中 $Z \sim N(0,1)$ 是标准正态分布

**步骤3：利用标准正态分布表**
- 标准正态分布的性质：
  - $P(Z > z) = 1 - \Phi(z)$
  - 其中 $\Phi(z)$ 是标准正态的累积分布函数
- 查表得：$\Phi(1) \approx 0.8413$
- 所以 $P(Z > 1) = 1 - 0.8413 = 0.1587$

**步骤4：得出结论**
- $P(X > 85) = P(Z > 1) = 0.1587$
- 即大约15.87%的学生成绩高于85分

#### 关键概念总结
- **正态分布 Normal Distribution** $X \sim N(\mu, \sigma^2)$
- **标准化 Standardization** $Z = \frac{X - \mu}{\sigma}$
- **标准正态分布 Standard Normal Distribution** $Z \sim N(0,1)$
- **累积分布函数 Cumulative Distribution Function** $\Phi(z) = P(Z \le z)$
- **对称性 Symmetry** 正态分布关于均值对称

# Q2

#### 问题翻译
某机器生产的零件长度服从正态分布 $N(20, 0.04)$（单位：厘米）。求：
1. 零件长度在19.5到20.5厘米之间的概率
2. 零件长度小于19.2厘米的概率
3. 零件长度大于20.8厘米的概率

#### 背景条件解释
**基本概念**：
- **正态分布 Normal Distribution** 对称的连续概率分布
- **均值 Mean** 分布的中心位置
- **标准差 Standard Deviation** 数据的离散程度
- **标准化 Z-score** 数据相对于均值的标准差数

**具体条件**：
- 零件长度分布：$X \sim N(\mu, \sigma^2)$
- 均值：$\mu = 20$ cm
- 方差：$\sigma^2 = 0.04$，所以标准差 $\sigma = 0.2$ cm
- 求解：不同长度区间的概率

#### 解答思路

**步骤1：确定分布参数**
- $X \sim N(20, 0.04)$
- 标准化公式：$Z = \frac{X - 20}{0.2}$

**步骤2：计算19.5到20.5厘米之间的概率**
- $P(19.5 < X < 20.5) = P\left(\frac{19.5-20}{0.2} < Z < \frac{20.5-20}{0.2}\right)$
- 计算Z值：
  - $Z_1 = \frac{19.5-20}{0.2} = \frac{-0.5}{0.2} = -2.5$
  - $Z_2 = \frac{20.5-20}{0.2} = \frac{0.5}{0.2} = 2.5$
- 所以 $P(-2.5 < Z < 2.5) = \Phi(2.5) - \Phi(-2.5)$
- 利用对称性：$\Phi(-2.5) = 1 - \Phi(2.5)$
- 查表：$\Phi(2.5) \approx 0.9938$
- $\Phi(-2.5) = 1 - 0.9938 = 0.0062$
- $P(-2.5 < Z < 2.5) = 0.9938 - 0.0062 = 0.9876$

**步骤3：计算小于19.2厘米的概率**
- $P(X < 19.2) = P\left(Z < \frac{19.2-20}{0.2}\right) = P(Z < -4)$
- $Z = \frac{19.2-20}{0.2} = \frac{-0.8}{0.2} = -4$
- $P(Z < -4) = \Phi(-4) \approx 0.00003$（非常小的概率）

**步骤4：计算大于20.8厘米的概率**
- $P(X > 20.8) = P\left(Z > \frac{20.8-20}{0.2}\right) = P(Z > 4)$
- $Z = \frac{20.8-20}{0.2} = \frac{0.8}{0.2} = 4$
- $P(Z > 4) = 1 - \Phi(4) \approx 1 - 0.99997 = 0.00003$（非常小的概率）

#### 关键概念总结
- **正态分布 Normal Distribution** $X \sim N(\mu, \sigma^2)$
- **标准化 Standardization** $Z = \frac{X - \mu}{\sigma}$
- **累积分布函数 CDF** $\Phi(z) = P(Z \le z)$
- **对称性 Symmetry** $\Phi(-z) = 1 - \Phi(z)$
- **标准正态分布 Standard Normal** $Z \sim N(0,1)$
