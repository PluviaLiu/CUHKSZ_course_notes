# Q1

#### 问题翻译
某设备故障时间服从参数为 $\lambda = 0.02$ 的指数分布，求该设备至少能正常运行40小时的概率。

#### 背景条件解释
**基本概念**：
- **指数分布 Exponential Distribution** 描述事件发生时间的连续分布
- **参数λ Lambda** 事件发生率（单位时间内发生的平均次数）
- **无记忆性 Memoryless Property** 过去不影响未来的特性
- **生存函数 Survival Function** 设备至少运行到某时间的概率

**具体条件**：
- 故障时间分布：$T \sim \text{Exp}(\lambda)$
- 参数：$\lambda = 0.02$（每小时故障率）
- 时间：$t = 40$ 小时
- 求解：$P(T > 40)$（至少运行40小时的概率）

#### 解答思路

**步骤1：理解指数分布**
- 指数分布的概率密度函数（PDF）：
  $$f(t) = \lambda e^{-\lambda t}, \quad t \ge 0$$
- 其中 $\lambda > 0$ 是参数
- 累积分布函数（CDF）：
  $$F(t) = P(T \le t) = 1 - e^{-\lambda t}$$

**步骤2：计算生存概率**
- 生存函数（Survival Function）：
  $$S(t) = P(T > t) = 1 - F(t) = e^{-\lambda t}$$
- 这直接给出了至少运行到时间t的概率

**步骤3：代入数值计算**
- 参数：$\lambda = 0.02$, $t = 40$
- 计算指数部分：
  $$\lambda t = 0.02 \times 40 = 0.8$$
- 计算生存概率：
  $$P(T > 40) = e^{-0.8}$$

**步骤4：计算最终结果**
- $e^{-0.8} \approx 0.4493$
- 所以 $P(T > 40) \approx 0.4493$
- 即大约44.93%的设备能至少正常运行40小时

#### 关键概念总结
- **指数分布 Exponential Distribution** $f(t) = \lambda e^{-\lambda t}$
- **参数λ Lambda** 事件发生率
- **累积分布函数 CDF** $F(t) = P(T \le t)$
- **生存函数 Survival Function** $S(t) = P(T > t) = e^{-\lambda t}$
- **无记忆性 Memoryless Property** $P(T > s + t | T > s) = P(T > t)$

# Q2

#### 问题翻译
顾客到达某服务台的间隔时间服从参数为 $\lambda = 0.1$ 分钟的指数分布。求：
1. 下一个顾客在5分钟内到达的概率
2. 下一个顾客在10分钟后才到达的概率
3. 已知已经等待了5分钟，下一个顾客在接下来的3分钟内到达的概率

#### 背景条件解释
**基本概念**：
- **指数分布 Exponential Distribution** 描述时间间隔的连续分布
- **到达率 Arrival Rate** 单位时间内到达的平均次数
- **间隔时间 Interarrival Time** 两次事件之间的时间间隔
- **无记忆性 Memoryless Property** 过去的等待时间不影响未来的等待时间

**具体条件**：
- 顾客到达间隔时间：$T \sim \text{Exp}(\lambda)$
- 参数：$\lambda = 0.1$ （每分钟到达率）
- 时间单位：分钟
- 求解：不同时间间隔的概率

#### 解答思路

**步骤1：确定分布参数**
- 间隔时间分布：$T \sim \text{Exp}(\lambda = 0.1)$
- 概率密度函数：$f(t) = 0.1 e^{-0.1t}$, $t \ge 0$
- 累积分布函数：$F(t) = P(T \le t) = 1 - e^{-0.1t}$
- 生存函数：$S(t) = P(T > t) = e^{-0.1t}$

**步骤2：计算5分钟内到达的概率**
- $P(T \le 5) = F(5) = 1 - e^{-0.1 \times 5} = 1 - e^{-0.5}$
- $e^{-0.5} \approx 0.6065$
- $P(T \le 5) = 1 - 0.6065 = 0.3935$
- 约39.35%的概率在5分钟内到达

**步骤3：计算10分钟后才到达的概率**
- $P(T > 10) = S(10) = e^{-0.1 \times 10} = e^{-1}$
- $e^{-1} \approx 0.3679$
- 约36.79%的概率在10分钟后才到达

**步骤4：应用无记忆性**
- 已知已经等待了5分钟，求在接下来的3分钟内到达的概率
- 无记忆性：$P(T \le 5 + 3 | T > 5) = P(T \le 3)$
- 这意味着过去的等待不影响未来的等待
- $P(T \le 3) = F(3) = 1 - e^{-0.1 \times 3} = 1 - e^{-0.3}$
- $e^{-0.3} \approx 0.7408$
- $P(T \le 3) = 1 - 0.7408 = 0.2592$
- 约25.92%的概率在接下来的3分钟内到达

#### 关键概念总结
- **指数分布 Exponential Distribution** $f(t) = \lambda e^{-\lambda t}$
- **无记忆性 Memoryless Property** 过去不影响未来
- **累积分布函数 CDF** $F(t) = P(T \le t) = 1 - e^{-\lambda t}$
- **生存函数 Survival Function** $S(t) = P(T > t) = e^{-\lambda t}$
- **到达率 Arrival Rate** $\lambda$ 单位时间内的平均到达次数
