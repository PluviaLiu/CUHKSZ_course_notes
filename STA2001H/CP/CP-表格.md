| 等公交车等多久               | 在区间$[a,b]$内**等可能发生**。<br>触发词: "uniformly distributed",<br>"randomly chosen from interval"                                    | **均匀分布** Uniform $U(a,b)$<br>👉 **支持集:** $x \in [a, b]$<br>$a$下限，$b$上限。<br><br>**核心推导：**<br>期望：$\mathbb{E}[X] = \int_a^b x \cdot \frac{1}{b-a} dx = \frac{1}{b-a} \cdot \frac{x^2}{2}\Big\rvert_a^b$<br>方差：$\mathbb{E}[X^2] = \int_a^b x^2 \cdot \frac{1}{b-a} dx = \frac{1}{b-a} \cdot \frac{x^3}{3}\Big\rvert_a^b$<br>MGF：$M(t) = \int_a^b e^{tx} \cdot \frac{1}{b-a} dx = \frac{1}{b-a} \cdot \frac{e^{tx}}{t}\Big\rvert_a^b$                                                | 密度函数(PDF): $f(x) = \frac{1}{b-a}$<br>求期望: $\mathbb{E}[X] = \frac{a+b}{2}$<br>求方差: $Var(X) = \frac{(b-a)^2}{12}$<br>求MGF: $M(t) = \frac{e^{tb}-e^{ta}}{t(b-a)}$ <br>MGF一阶导: $M'(t) = \frac{t(be^{tb}-ae^{ta}) - (e^{tb}-e^{ta})}{t^2(b-a)}$              |

# 3️⃣ PDF - CDF 与联合分布转换公式 (1D vs 2D)

| **特性 / 动作**                      | **一维单变量 (1D)**                                                     | **二维联合分布 (2D Joint)**                                                                                                                                                        |
| :------------------------------- | :----------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **符号与直观意义**                      | **PDF:** $f(x)$ (曲线高度)<br>**CDF:** $F(x)$ (曲线下面积)                  | **PDF:** $f(x,y)$ (3D曲面高度)<br>**CDF:** $F(x,y)$ (曲面下的体积)                                                                                                                     |
| **CDF 定义**                       | $F(x) = P(X \le x)$                                                | $F(x,y) = P(X \le x, Y \le y)$                                                                                                                                               |
| **🔥 范围限制 (Limits)**             | **PDF:** $f(x) \ge 0$ (⚠️**可以大于1**)<br>**CDF:** $0 \le F(x) \le 1$ | **PDF:** $f(x,y) \ge 0$ (⚠️**可以大于1**)<br>**CDF:** $0 \le F(x,y) \le 1$                                                                                                       |
| **PDF $\to$ CDF (积分)**           | $F(x) = \int_{-\infty}^{x} f(t) dt$                                | $F(x,y) = \int_{-\infty}^{y} \int_{-\infty}^{x} f(u,v) du dv$                                                                                                                |
| **CDF $\to$ PDF (求导)**           | $f(x) = \frac{d}{dx}F(x)$                                          | $f(x,y) = \frac{\partial^2}{\partial x \partial y}F(x,y)$ (先对x偏导，再对y偏导)                                                                                                      |
| **全概率归一 (求常数c)**                 | $\int_{-\infty}^{\infty} f(x) dx = 1$                              | $\int_{-\infty}^{\infty} \int_{-\infty}^{\infty} f(x,y) dx dy = 1$                                                                                                           |
| **求边缘分布 (Marginal)**             | N/A (本身就是一维)                                                       | $f_X(x) = \int_{-\infty}^{\infty} f(x,y) dy$ <br>💡**口诀：求谁就不积谁** (求X的边缘，就把y积分积掉)                                                                                             |
| **求条件分布 (Conditional)**          | N/A                                                                | $f(x\|y) = \frac{f(x,y)}{f_Y(y)}$ (前提: $f_Y(y) > 0$)<br>$f(y\|x) = \frac{f(x,y)}{f_X(x)}$ (前提: $f_X(x) > 0$)<br>💡**口诀：条件在下，联合在上**                                           |
| **🔥求条件期望 (Cond. Exp)**          | N/A                                                                | $\mathbb{E}[X\|Y=y] = \int_{-\infty}^{\infty} x \cdot f(x\|y) dx$<br>$\mathbb{E}[Y\|X=x] = \int_{-\infty}^{\infty} y \cdot f(y\|x) dy$<br>⚠️**注意：** 积分里用的是**条件PDF**，不是联合PDF！ |
| **🔥 独立性检验 (Independence)**      | N/A                                                                | **若独立，必须满足：** $f(x,y) = f_X(x) \cdot f_Y(y)$<br>*(对所有x,y都成立，且积分范围必须是矩形)*                                                                                                     |
| **🔥 联合期望 (求 $\mathbb{E}[XY]$)** | $\mathbb{E}[g(X)] = \int g(x)f(x)dx$                               | $\mathbb{E}[XY] = \int_{-\infty}^{\infty} \int_{-\infty}^{\infty} xy \cdot f(x,y) dx dy$<br>*(算出这个才能求协方差 $Cov(X,Y)$)*                                                        |
| **🔥 全期望/全方差公式**                 | N/A                                                                | **全期望:** $\mathbb{E}[X] = \mathbb{E}[\mathbb{E}[X \|Y]]$<br>**全方差:** $Var(X) = \mathbb{E}[Var(X\|Y)] + Var(\mathbb{E}[X\|Y])$                                                |

我把这三块拼图给你补上了。这下这个表真的是**“二维连续型随机变量”的终极无死角版本**了：

如果题目让你判断 $X$ 和 $Y$ 是否独立，**先看积分范围（支持集）！**
如果 $x$ 和 $y$ 的范围是互相牵制的（比如 $0 < x < y < 1$），**直接秒杀：绝对不独立！** 连算都不用算。只有当范围是矩形（比如 $0<x<1, 0<y<2$），才需要去验证 $f(x,y) = f_X(x)f_Y(y)$。

# 4️⃣计算结论

| **类型**   | **积分式**                                                   | **结果**               | **常见用途**       |
| -------- | --------------------------------------------------------- | -------------------- | -------------- |
| **基础指数** | $\int_{0}^{\infty} e^{-ax} dx$                            | $\frac{1}{a}$        | 指数分布全概率归一化     |
| **一阶期望** | $\int_{0}^{\infty} x e^{-ax} dx$                          | $\frac{1}{a^2}$      | 指数分布的期望 $E[X]$ |
| **二阶矩**  | $\int_{0}^{\infty} x^2 e^{-ax} dx$                        | $\frac{2}{a^3}$      | 计算方差 $Var(X)$  |
| **高斯积分** | $\int_{-\infty}^{\infty} e^{-x^2} dx$                     | $\sqrt{\pi}$         | 正态分布的基础        |
| 指数函数     | $\sum_{k=0}^{\infty} \frac{a^k}{k!}$                      | $e^a$                | 化简求和式子         |
| 伽马函数     | $\int_{0}^{\infty} x^n e^{-ax} dx$                        | $\frac{n!}{a^{n+1}}$ |                |
| 尾概率      | $P(X > n) = \sum_{x=n+1}^{\infty} (1-\theta)^{x-1}\theta$ | $(1-\theta)^n$       | 知道n+1次或更晚才发生   |
# 5️⃣ i.i.d数据表

| **特性 / 统计量** | **均值 $\bar{X}_n$**                   | **最大值 $Y_n$**                                              | **最小值 $Z_n$**                           |
| :----------- | :----------------------------------- | :--------------------------------------------------------- | :-------------------------------------- |
| **数学表达**     | $\bar{X}_n = \frac{1}{n}\sum X_i$    | $Y_n = \max\{X_1,\dots,X_n\}$                              | $Z_n = \min\{X_1,\dots,X_n\}$           |

| **PDF**      | *(无通用公式，依赖原分布)*                      | $f_{Y_n}(x) =\mathbb{P}(Y_n \leq x)$$=1 - [1-F(x)]^n$      | $f_{Z_n}(x) = n[1 - F(x)]^{n-1}f(x)$    |
| **本身的问题**    | 趋向常数 $\mu$（WLLN）                     | 趋向 $+\infty$                                               | 趋向 $-\infty$（或0）                        |
| **怎么拉回来**    | 减 $\mu$，再除以 $\sigma/\sqrt{n}$（CLT方法） | 减 $\log n$                                                 | 乘以 $n$                                  |
| **极限分布**     | $N(0,1)$                             | Gumbel，$e^{-e^{-x}}$                                       | 指数分布                                    |
