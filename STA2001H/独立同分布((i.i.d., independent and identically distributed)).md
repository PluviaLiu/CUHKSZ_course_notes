#### 1. 定义是什么？
想象你在**反复扔同一枚硬币**：
- 每次扔都**不受上一次影响**（独立）
- 每次扔的硬币**是同一枚**，正面概率永远是 50%（同分布）

[[独立 (Independent)]]
$$\mathbb{P}(X_1 \in A_1, X_2 \in A_2, \dots, X_n \in A_n) = \prod_{i=1}^n \mathbb{P}(X_i \in A_i)$$
[[同分布 (Identically Distributed)]]
$$F_{X_1}(x) = F_{X_2}(x) = \cdots = F_{X_n}(x) = F(x), \quad \forall x$$

#### 2. i.i.d衍生的三大统计量
已知，有 $n$ 个 i.i.d. 随机变量 $X_1, X_2, \dots, X_n$。

问题是：$\text{当 } n \to \infty,\quad Y_n \text{ 这个随机变量的分布长什么样？}$（抽的人越多，最大值只会增不会减，所以 $Y_n$ 本身趋向 $+\infty$，没有极限分布。所以需要变换）

具体例子：（原来的量**减去某个东西**之后，极限分布是一个固定的形状）
$$\underbrace{\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}}}_{\text{平均值标准化}} \xrightarrow{d} N(0,1) \qquad \text{vs} \qquad \underbrace{Y_n - \log n}_{\text{最大值平移}} \xrightarrow{d} \text{Gumbel}$$

| **特性 / 统计量** | **均值 $\bar{X}_n$** | **最大值 $Y_n$** | **最小值 $Z_n$** |
| :--- | :--- | :--- | :--- |
| **数学表达** | $\bar{X}_n = \frac{1}{n}\sum X_i$ | $Y_n = \max\{X_1,\dots,X_n\}$ | $Z_n = \min\{X_1,\dots,X_n\}$ |
| **CDF** | *(无通用公式，依赖原分布)* | $F_{Y_n}(x)= \mathbf{[F(x)]^n}$ | $F_{Z_n}(x)= \mathbf{1 - [1 - F(x)]^n}$ |
| **PDF** | *(无通用公式，依赖原分布)* | $f_{Y_n}(x) = n[F(x)]^{n-1}f(x)$ | $f_{Z_n}(x) = n[1 - F(x)]^{n-1}f(x)$ |
| **本身的问题** | 趋向常数 $\mu$（WLLN） | 趋向 $+\infty$ | 趋向 $-\infty$（或0） |
| **怎么拉回来** | 减 $\mu$，再除以 $\sigma/\sqrt{n}$（CLT方法） | 减 $\log n$ | 乘以 $n$ |
| **极限分布** | $N(0,1)$ | Gumbel，$e^{-e^{-x}}$ | 指数分布 |

n趋于极限时的两大极限方向


#### 3. 当样本量极大 ($n \to \infty$) 时会发生什么？(极限理论)
既然是 i.i.d.，当抽样人数 $n$ 趋向无穷大时，统计量会呈现出极其规律的**极限行为**。主要分为两个方向：

**方向一：研究“均值 $\bar{X}_n$”的极限**
*   **弱大数定律 (WLLN)**：描述“值”的收敛。样本均值最终会死死钉在一个**常数** $\mu$ 上。（依概率收敛）
*   **中心极限定理 (CLT)**：描述“形状”的收敛。把均值标准化 $\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}}$ 后，它的**分布形状**完美趋近于标准正态分布 $N(0,1)$。（依分布收敛）

**方向二：研究“极值 $Y_n, Z_n$”的极限 (极值理论 EVT)**
极值标准化后，不会变成正态分布，而是会趋向特定的“极值分布”。（注意：不是所有分布都有极值分布，只对特定类型成立）
*   **Gumbel 分布**：CDF 趋向 $e^{-e^{-x}}$。
    *   **触发条件**：原分布的尾部是无限延伸的（无上界）。
    *   **常见例子**：正态分布、指数分布、Logistic分布。
*   **Weibull 分布**：CDF 趋向 $e^{-(-x)^\alpha}$ (当 $x<0$ 时)。
    *   **触发条件**：原分布有严格的有限上界（比如最大只能取到 1）。
    *   **常见例子**：均匀分布 $U(0,1)$。

#### 4. 考试实战：如何求极值的极限分布？(标准做题 SOP)
考试时，题目通常会**直接给你原分布的 $F(x)$**，让你自己推导最大值 $Y_n$ 经过变换后的极限分布。

**🔥 闭眼按以下 4 步走：**

1. **写出最大值的精确 CDF**
   * 直接套公式：$F_{Y_n}(y) = [F(y)]^n$
2. **代入题目给的变换 (平移/缩放)**
   * 题目通常会定义一个新变量，比如 $U_n = Y_n - \log n$。
   * 把 $Y_n$ 替换掉：求 $\mathbb{P}(U_n \le u) = \mathbb{P}(Y_n - \log n \le u) = \mathbb{P}(Y_n \le u + \log n)$。
   * 代入第一步的式子：$F_{U_n}(u) = [F(u + \log n)]^n$。
3. **求极限 (核心计算)**
   * 对上式取极限：$\lim_{n\to\infty} [F(u + \log n)]^n$。
   * *高数技巧预警*：这里通常会遇到 $1^\infty$ 型极限，必须用重要极限公式 $\lim_{n\to\infty} (1 + \frac{x}{n})^n = e^x$ 来化简！
4. **收尾与命名 (Bonus)**
   * 验证算出来的极限 CDF 是否合法（单调递增，且在 $0$ 到 $1$ 之间）。
   * 认出它叫什么名字（比如算出来是 $e^{-e^{-u}}$ 就写 Gumbel，加分项，不认识写出式子也给绝大部分分）。



- **WLLN**：平均值会收敛到一个**常数** $\mu$
- **CLT**：标准化后的均值，**分布**趋向正态 $N(0,1)$
- **Gumbel**：$Y_n$ 做变换后，它的 **CDF** 趋向 $e^{-e^{-x}}$，即最大值的**分布形状**趋向 Gumbel 分布。例子：正态分布、指数分布、**Logistic分布**
- **Weibull** ： $e^{-(-x)^\alpha}$ ，有有限上界，例子：均匀分布


#### 4. 考试时可以干什么？

- **大数定律 (Law of Large Numbers)**：样本均值趋近总体均值
- **中心极限定理 (Central Limit Theorem)**：样本均值标准化后趋近正态分布


不是通用的，**只对特定类型的分布成立**。

考试会**直接给你 $F(x)$**，让你自己算极限：

● 代入 $F_{Y_n}(x) = [F(x)]^n$

↓ 做变换（比如 $U_n = Y_n - \log n$）

↓ 算 $\lim_{n\to\infty}[F(x + \log n)]^n$

↓ 得到极限 CDF，验证合法性

↓ 认出它叫什么名字（加分，不认识也没关系）
